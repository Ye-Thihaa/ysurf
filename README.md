# Portfolio — Ye Thihaa

Personal portfolio website built from scratch while learning frontend development in 2025.

---

## Tech Stack

- **UI** — React, Tailwind CSS
- **Animations** — Custom `FadeInSection` scroll reveals
- **Deployment** — Vercel

## Features

- Branching timeline on desktop, clean prose on mobile
- Scroll progress indicator
- Light / dark theming
- Resume download
- Fully responsive

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:8469](http://localhost:8469) in your browser.

## Project Structure

```
├── components/
│   ├── Layout.tsx
│   ├── FadeInSection.tsx
│   └── ScrollProgress.tsx
├── pages/
│   └── about.tsx
├── public/
│   └── resume.pdf
└── styles/
```

## Deployment

Deployed on [Vercel](https://vercel.com). Any push to `main` triggers a new deployment.
